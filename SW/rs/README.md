# rs

Generated by Hulotte framework.

## Build

```bash
./build.sh
```

Or manually: (but before you need to copy them in your hulotte directory then you need to change the cmake manually)

```bash
mkdir build && cd build
cmake .. \
    -DSTREAMPU_ROOT=/net/netud/e/nhitar/Bureau/hulotte/streampu \
    -DAFF3CT_ROOT=/net/netud/e/nhitar/Bureau/hulotte/aff3ct \
    -DHULOTTE_ROOT=/net/netud/e/nhitar/Bureau/hulotte
make -j
```

## Run

```bash
./build/rs
```
