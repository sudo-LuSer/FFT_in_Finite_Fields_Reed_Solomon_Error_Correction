# rs

Generated by Hulotte framework.

## Build

```bash
./build.sh
```

Or manually: (but before you need to copy them in your hulotte directory then you need to change the cmake manually)

```bash
mkdir -p build && cd build

cmake .. \
  -DSTREAMPU_ROOT=/chemin/vers/streampu \
  -DAFF3CT_ROOT=/chemin/vers/aff3ct \
  -DHULOTTE_ROOT=/chemin/vers/hulotte

make -j$(nproc)
```

## Run

```bash
./build/rs
```
